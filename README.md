### Description

A set of simple games

### Setup

```sh
$ make install
```

### 1 brain-even.

Answer if the number is even or not.

### Game launch

```sh
$ brain-even
```

[![asciicast](https://asciinema.org/a/385963.svg)](https://asciinema.org/a/385963)
